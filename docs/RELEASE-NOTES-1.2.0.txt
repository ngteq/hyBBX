HyBBX v1.2.0
============

Multi-transport session daemon: telnet (:2323), SSH (:3232), WebSocket
(loopback :4591). Standalone clients: hybbx-telnet, hybbx-ssh, hybbx-terminal.

What's new
----------
  hybbx-ssh client (libssh; no host-key prompt)
  hybbx-telnet AmigaOS cross-build (bebbo-gcc; verified A1200 / AmigaOS 3.2)
  Command history (25 entries) in sessions and native clients
  AX.25 auto-broadcast: 1200 baud UI, max 48 chars, min 5 min interval
  WebSocket idle ping keepalive; browser auto-reconnect

Clients
-------
  hybbx-telnet   — Linux/POSIX; AmigaOS via build-amiga-telnet.sh
  hybbx-ssh      — encrypted HyBBX SSH transport
  hybbx-terminal — HBX circuit client

Broadcast
---------
  [broadcast] ax25_auto, ax25_auto_interval (min 300), ax25_auto_message
  Default: Broadcast: @service@ online

Status
------
  Verified : telnet, WebSocket, AmigaOS hybbx-telnet
  Built    : SSH, hybbx-ssh, AX.25 auto-broadcast, packet_radio, baycom (opt-in)
  Partial  : TCP /broadcast fan-out (stub)

Build
-----
  cmake -B build -DCMAKE_BUILD_TYPE=Release && cmake --build build
  ./scripts/build-clients.sh
  ./scripts/build-amiga-telnet.sh  (bebbo-gcc at /opt/amiga)

Docs
----
  Operator documentation assumes Linux (HTTPD_DOCROOT, systemctl, ss).

Demo
----
  Browser : https://hybbx.un1t.me/
  Telnet  : telnet hybbx.un1t.me 2323
  SSH     : ssh hybbx.un1t.me -p 3232
