HyBBX v1.5.0 release notes

Topology: Main, Secondary, proxy network (mains_proxy).
HBX/Circuit sole inter-node transport on :7323.

New / updated:
- Command registry (share/commands.yaml) and help layout (COMMANDS.md)
- /help, /menu, /index, /alias — compact 2400-baud menus
- /broadcast and /announce — Sysop message to all online users on local Main
- AX.25 auto-beacon (INI ax25_auto only; not the /broadcast command)
- mains_proxy plugin (proxy network; HYBBX_PLUGIN_MAINS_PROXY)
- /proxymail and /proxychat — delivery not available yet
- Built-in [security] (network + abuse), CALLID bans
- SQLite storage opt-in, hybbx-ssh client, command history

User groups: Sysop, Admin, Mod, User, Guest (no Staff term).

Transports: telnet, SSH, WebSocket, HBX circuit,
packet_radio, mains_proxy (partial), ARDOP, CRDOP, BayCom (opt-in).

See docs/RELEASE-1.5.0.md, docs/TOPOLOGY.md, docs/FEATURES.md.
