HyBBX v1.1.1
============

Multi-transport session daemon: telnet (:2323), SSH (:3232), WebSocket
(loopback :4591). Mail, chat, and / commands on every path.

What's new
----------
  %time% and %date% tokens in text/*.txt (banner, motd, news, rules)
  [time] clock, seconds (default yes), date (iso | iso_short | us | eu)
  Refreshed default text files

Transports
----------
  Telnet        :2323 — primary verified path
  SSH           :3232 — libssh; wire auth separate from HyBBX accounts
  WebSocket     :127.0.0.1:4591/hybbx — reverse-proxy + httpd docroot UI
  packet_radio  — multi-TNC edge; kiss_entry/exit; TNCS.md
  baycom        — opt-in BayCom PR-Stack (HYBBX_PLUGIN_BAYCOM=OFF default)
  ARDOP / CRDOP — external modem host clients

RF / edge
---------
  Up to 8 packet_radio instances per Secondary
  TNC2C 7E1 + RTS/DTR; tnc2, pk232, mfj1278, kantronics profiles
  HBX AX.25 bridge to Main

Status
------
  Verified : telnet, mail, chat, commands, text time/date tokens
  Built    : SSH, WebSocket, packet_radio, baycom (opt-in), ARDOP, CRDOP

Build
-----
  cmake -B build -DCMAKE_BUILD_TYPE=Release && cmake --build build
  libssh required for SSH plugin (default ON)
  Full-featured: -DHYBBX_PLUGIN_BAYCOM=ON
  AmigaOS 3.9+ cross-GCC: see docs/BUILD.md

Docs
----
  Operator documentation assumes Linux (HTTPD_DOCROOT, systemctl, ss).

Demo
----

  Telnet  : telnet hybbx.un1t.me 2323
  SSH     : ssh hybbx.un1t.me -p 3232
